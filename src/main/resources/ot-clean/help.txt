ot-clean

-d      - By default no directories are going to be deleted. You have to provide this argument
          if you really want to delete them.
-r      - By default application is going to clean garbage from current
          directory only. If you want to search nested directories, use -r

Example usage: /> ot-clean -dr

You can create example file called [clean.yml].

Example file content:
deletes:
 - this file.txt
 - that directory
